================================================================================
        APPLIED PROBABILITY FRAMEWORK - WINDOWS EXECUTABLE BUILDER
================================================================================

QUICK START:
------------
1. Double-click "build_exe.bat" (easiest method)
   OR
2. Open Command Prompt and run: python build_exe.py

The executable will be created in: dist\AppliedProbabilityFramework\

WHAT YOU GET:
-------------
- AppliedProbabilityFramework.exe - Standalone executable
- All necessary dependencies bundled
- No Python installation required to run!

USAGE:
------
After building, navigate to dist\AppliedProbabilityFramework\ and run:

    AppliedProbabilityFramework.exe --help
    AppliedProbabilityFramework.exe run
    AppliedProbabilityFramework.exe run --num-simulations 10000

DISTRIBUTION:
-------------
Zip the entire "AppliedProbabilityFramework" folder and share it.
Users just need to extract and run the .exe file!

For detailed instructions, see BUILD_EXE_INSTRUCTIONS.md
